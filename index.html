What is Shaven?

Shaven is a DOM building utility and template engine based upon JsonML. This simple markup language lets you easily represent XML with JSON. Templates are therefore simply Javascript arrays.

Installation


        
On the server you can install Shaven with npm:

        
$ npm install shaven


        
Then require it in your scripts like this:

        
var shaven = require('shaven')


        
In the browser just load the bundled javascript file
            (get it from github):

        
<script src="shaven.js"></script>


        
Now you can simply pass a JsonML-array to Shaven:

        
var shavenObject = shaven(['p', 'some text'])
